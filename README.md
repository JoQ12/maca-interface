# maca-interface
MACA est une place de marché numérique de type Bourse de Matières Premières Agricoles (Commodities Exchange) conçue pour le contexte africain. Elle transforme le commerce de gros — aujourd’hui fragmenté dans des cercles privés et opaques — en un écosystème ouvert, transparent et sécurisé. La solution permet de connecter les grands producteurs
